# 🌙 Site de Perfil: Sailor Moon (HTML & CSS)

Este projeto representa minha primeira tentativa mais séria de desenvolver um site de perfil de apresentação utilizando HTML e CSS. O objetivo foi criar uma página estática com informações básicas, uma imagem de perfil, descrição e botões/links que direcionam para as redes sociais da "cliente" (neste caso, a personagem Sailor Moon, usada como tema para o estudo).

---

## 🧠 Objetivo do Projeto

-   **Praticar HTML:** Estruturar uma página web completa, utilizando tags semânticas e organizando o conteúdo.
-   **Dominar CSS:** Aplicar estilos para criar um design visualmente atraente, responsivo (se aplicável) e alinhado ao tema proposto.
-   **Desenvolvimento de Layout:** Experimentar com diferentes propriedades de CSS para posicionamento, cores, fontes e interatividade básica (efeitos de hover nos botões).
-   **Integração de Mídias:** Inserir imagens e configurar links externos para redes sociais.
-   **Simular um Cliente:** Trabalhar com um "briefing" (mesmo que imaginário) para entregar uma solução visual.

---

## 🛠️ Tecnologias Utilizadas

-   **Linguagens:** HTML, CSS
-   **Ferramentas:** VS Code, Git, GitHub

---

## ✨ Funcionalidades

-   Página de apresentação com imagem de perfil.
-   Nome e descrição da "cliente" (Sailor Moon).
-   Botões interativos que simulam links para redes sociais.
-   Estilização personalizada com tema visual da Sailor Moon.

---

## ▶️ Como Visualizar o Projeto

Para ver este site no seu navegador, siga os passos abaixo:

1.  **Clone o repositório:**
    Abra seu terminal ou prompt de comando e execute:
    ```bash
    git clone https://github.com/WillsKFG/Site2.Perfil-Sailor-Moon.git
    ```

2.  **Navegue até o diretório do projeto:**
    ```bash
    cd Site2.Perfil-Sailor-Moon
    ```

3.  **Abra o arquivo HTML no navegador:**
    Localize o arquivo `Site pessoal da Sailor Moon.html` na pasta do projeto e abra-o diretamente com seu navegador de preferência (clicando duas vezes ou arrastando para a janela do navegador).
